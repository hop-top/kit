## User story

**As a** `kit-contributor`, **I want** to run a single command
(`nix develop`, `devbox shell`, or open in VS Code) to get a fully
configured dev environment, **so that** I can start contributing
without manual tool installation.

## Acceptance criteria

- [ ] `nix develop` provides Go 1.26, Node 22, Python 3.13, all
  linters, go-task, git, gh CLI
- [ ] `devbox shell` produces identical environment
- [ ] VS Code Dev Containers opens with Go, TS, Python extensions
- [ ] GitHub Codespaces works without additional setup
- [ ] `task check` passes inside the dev environment

## Notes

Three entry points (nix, devbox, devcontainer) all consume the
same `flake.nix` source of truth.
