## User story

**As a** `cli-author`, **I want** to scaffold a new CLI with one
function call, **so that** all standard flags and behaviours work out
of the box.

## Acceptance criteria

- [ ] Go: `cli.New(cli.Config{...})` returns root with all standard
  flags
- [ ] TS: `createCLI({...})` returns Commander program with all
  standard flags
- [ ] Python: `create_app(...)` returns Typer app with all standard
  flags
- [ ] No manual flag registration needed for --format, --quiet,
  --no-color, --version

## Notes

Already implemented in kit v0.1. Story captures existing behaviour
for traceability.
