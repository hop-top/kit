## User story

**As a** `kit-contributor`, **I want** to install AI coding tools
(Claude Code, Copilot, Codex, Gemini, etc.) via an interactive
menu or auto-setup file, **so that** I can use my preferred tools
without polluting the base dev image.

## Acceptance criteria

- [ ] `task ai:setup` shows interactive multi-select
- [ ] Individual `task ai:<tool>` installs specific tool
- [ ] Each install script is idempotent (skip if present)
- [ ] `.ai-tools` file triggers auto-install during
  `post-create.sh`
- [ ] AI tools never installed in CI
